Horizons setup

Want to connect to the server? Follow this guide. If you run into any problems, join the Discord!

Finding your config.ini

The server you connect to is saved in a configuration file, and to change that, you need to find the file!
Windows:

  1. Open Explorer and navigate to C:\Users\(user)\AppData\Local\IWM (AppData is a hidden folder, enable hidden folders!)
  2. Your config.ini will be there

Linux:

  1. Open your file manager
  2. Navigate to /home/(user)/.local/share/Steam/steamapps/compatdata
  3. Look through all the folders here until you find one with IWM in pfx/drive_c/users/steamuser/AppData/Local/ OR search for the "IWM" folder
  4. Your config.ini file will be in threre

Changing the server (method 1)

Download the latest version of the IWM server replacer for your operating system.
Run it (if on mac or linux do so via the command line with ./app-name). Allow it to create a config file and give it the location of your config.ini file.
Select option 2 and enter https://horizons.coding398.dev with no slash at the end, then select option 3 to save and exit.
Close and open I Wanna Maker.


Changing the server (method 2)

Open config.ini with a text editor and scroll down until you see Server_hostname and Server_hostname_fallback.

Replace both with exactly "https://horizons.coding398.dev", do not add a http:// one.

Close and open I Wanna Maker.


Check that it's working

When you enter the game, the amount of levels that show should be different.
To be sure you're connected, enter a random login into the prompt, and you should get a link to https://horizons.coding398.dev/login.


Create an account and hop in!

Go to horizons.coding398.dev/login and log-in with your Discord account.
You'll be given a username and password you can use in-game to log in!
Once you're in, you can start playing with no rules!


Cross-compatibility with the main server

To save you hassle, you can connect to the main server via Horizons!
Just log out and log in with the credentials to your official IWM server account.
The proxy used will also make it look like everyone, including you, has admin! You can get special perks with that little trick.
